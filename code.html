Free Software

I am devoted to broadening the availability of research tools used by students and researchers to analyze their data. Although the scientific community embraces publicly released data and open-access publications, the code and software that researchers use to determine their findings is largely hidden and proprietary. I believe that free and open-source software benefits the entire scientific community, empowers the public, and makes results more easily independently verifiable.

My Code

You can find my code released under the GNU General Public License on my github page largely in my personal library besl. I do most of my code in python but also have experience with IDL and C. I have working knowledge of perl, fortran 77, golang, and smalltalk.
